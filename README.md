# Customer and Order Management Analysis

##  Project Overview
This project focuses on analyzing customer data and order transactions to extract meaningful business insights. The analysis includes customer behavior, revenue trends, city-wise performance, and high-value customer identification.

##  Tools Used
- SQL
- Excel
- Data Analysis Techniques
- Window Functions

##  Key Analysis Performed

1. Customer and Order Listing
2. Customers by City
3. Total Revenue Generated
4. Average Order Value
5. Total Spending per Customer
6. Customers with No Orders
7. Monthly Order Revenue
8. Highest Order Amount
9. Orders over $150
10. Top 5 Highest Spending Customers
11. Ranking Customers by Spending (Window Function)
12. Running Total of Order Amounts

##  Business Insights

- Identified high-value and premium customers.
- Analyzed revenue contribution by city.
- Found inactive customers representing missed revenue opportunities.
- Observed monthly revenue spikes driven by high-value orders.
- Used ranking and cumulative revenue techniques for deeper insights.

##  Key Learnings

- SQL Joins for combining customer and order data
- Aggregate functions (SUM, AVG, COUNT)
- Window functions (RANK, Running Total)
- Revenue trend analysis
- Customer segmentation
